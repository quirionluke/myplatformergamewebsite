Post Mortem

If there were more time to work on this project, I would have added more polish, probably gotten the sound effects to be a little snappier. I would have added more levels to the game and made it overal run smoother. I would have implemented the localization, which is something that I could not figure out in the time we had to do the assignment. I also would have better documented my code, as it was something that I maybe should have done as I went along instead of trying to put into the code once I was sure it all worked. I also would have done a better job on the trailer and the screenshots. Overall with all of the things that was necessary to do for this assignment and all of the things I had to do in the same time for other classes, I could not devote as much time as I would have liked to on this assignment and was unable to complete all of the parts in just two weeks while also keeping up with the other labs in this course and the other homework for my other courses. I probably could also make an initial commit from the beginning so that I can refer to the starter code as I try to get functional code to commit. This was a workflow issue I encountered where I began the assignment when it was assigned, but didn't commit until I had a functional breakout game. In making that version of the game function, I probably should have kept version control through git so I could revert to earlier editions of the game when I was having issues, and that probably led to a lot of slowdown. For the next project I'll do some more googling to try and find helpful tutorials for the parts of the assignment that might not be covered in class, as well as asking my peers who have struggled on it for their input on how they got past certain problems. I could probably have attended office hours when I was struggling as well, but most likely I could spend more time on the next assignment and less time on other things. If I were to build different tools, I probably would have made a levels class, and maybe introduced some powerups into the game, with their own class. I also could randomize the colors of the blocks because they came from the tutorial I found in a spritesheet with multiple colors.
